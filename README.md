# 📝 iNotes - PHP CRUD App

## 📌 Overview
iNotes is a simple CRUD (Create, Read, Update, Delete) web application built using PHP and MySQL. It allows users to manage notes efficiently with a clean UI.

---

## 🚀 Features
- ➕ Add new notes
- 📖 View all notes
- ✏️ Edit existing notes (modal-based)
- ❌ Delete notes with confirmation
- 🔍 Search & sort using DataTables
- ✅ Input validation (min length check)
- 🔒 Basic security using escaping (mysqli_real_escape_string)

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Bootstrap
- **Backend:** PHP
- **Database:** MySQL
- **Library:** jQuery, DataTables
